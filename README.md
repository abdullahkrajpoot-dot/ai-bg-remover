# ✂️ AI Background Remover

Ye aik behtareen Python-based tool hai jo AI ki madad se kisi bhi image ka background sirf aik click mein saaf kar deta hai. Is project mein **Streamlit** ko frontend aur **rembg** library ko background removal ke liye istemal kiya gaya hai.

 🚀 Features
 **Easy Upload:** Apni photo select karein (JPG, JPEG, PNG).
 **AI Power:** Deep learning model khud hi background detect kar ke mita deta hai.
 **High Quality:** Resulting image transparent background ke sath milti hai.
 **Instant Download:** Transparent image ko foran download karein.

 🛠️ Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **Library:** Rembg (AI Model)
* **Image Processing:** PIL (Pillow)

💻 Installation & Setup

1.  **Repository Clone karein:**
    bash
    git clone [https://github.com/abdullahkrajpoot-dot/ai-bg-remover.git](https://github.com/abdullahkrajpoot-dot/ai-bg-remover.git)
    cd ai-bg-remover

2.  **Requirements Install karein:**
    bash
    pip install streamlit rembg pillow

3.  **App Run karein:**
    bash
    streamlit run bgremover.py
