# ✂️ AI Background Remover

A fast and efficient web application to remove image backgrounds automatically using Artificial Intelligence. Built with **Python**, **Streamlit**, and the **Rembg** library.

## 🚀 Features
* **Instant Removal:** Upload an image and get a transparent background in seconds.
* **Side-by-Side Comparison:** View the original and processed images together.
* **High Quality:** Uses the `u2net` AI model for precise edge detection.
* **Easy Download:** Save your processed image as a transparent PNG.

## 🛠️ Tech Stack
* [Python](https://www.python.org/)
* [Streamlit](https://streamlit.io/)
* [Rembg](https://github.com/danielgatis/rembg)
* [Pillow (PIL)](https://python-pillow.org/)

## 💻 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/abdullahrajpoot-dot/AI-BG-REMOVER.git](https://github.com/abdullahrajpoot-dot/AI-BG-REMOVER.git)
   cd AI-BG-REMOVER

2. Install Dependencies:
  Bash
  pip install streamlit rembg pillow "onnxruntime"

3.Run the App:
  Bash
  streamlit run bgremover.py
